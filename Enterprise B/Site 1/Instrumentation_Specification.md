## 2. INSTRUMENT TAG NUMBERING CONVENTION

### 2.1 Tag Number Format

Per ISA-5.1, instrument tags follow this format:

```
[First Letter][Succeeding Letters]-[Area]-[Equipment]-[Loop Number]
```

**First Letter (Measured Variable):**
| Letter | Variable |
|--------|----------|
| F | Flow |
| T | Temperature |
| W | Weight |
| L | Level |
| P | Pressure |
| A | Analytical |
| S | Speed |
| Z | Position/State |

**Succeeding Letters (Function):**
| Letter | Function |
|--------|----------|
| T | Transmitter |
| I | Indicator |
| C | Controller |
| V | Valve |
| S | Switch |
| E | Element |
| Y | Relay/Compute |
| A | Alarm |

**Area Codes:**
| Code | Area |
|------|------|
| LP | Liquid Processing |
| FP | Filler Production |
| PK | Packaging |
| PZ | Palletizing |
| UT | Utilities |

- `FT-LP-VAT01-001` = Flow Transmitter, Liquid Processing, Vat 01, Loop 001

All calibration equipment shall be traceable to NIST standards. Calibration records shall include:
- Date of calibration
- Technician performing calibration
- As-found and as-left data
- Reference standard identification
- Next due date
